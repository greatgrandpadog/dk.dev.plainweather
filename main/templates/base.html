Plain Weather

This Plain Weather web application provides simple weather and forecast information using the unofficial Google Weather API. It makes use of Python (v2.7), web.py (v3.2), SqlAlchemy(v0.6.8), Mako (v0.3.6), Mootools (v1.4.0.1), and JSON. The application was developed using Apache to serve the content via mod_wsgi. It was designed to work with Firefox 7 and Chrome 15.


Source code is available as a tar.gz file (here), or from a github repository (here).
